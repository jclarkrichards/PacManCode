Phase I

I divided my process into three phases. The first phase focuses on getting everything set up and getting a very basic version of Pacman running. The second phase will get the entire Pacman game running, and the third phase is all about polishing the game like adding nice graphics. In order to get the most out of all of this I would highly suggest you take each section in order. The sections build upon each other, so try not to skip around.
