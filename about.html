About Me

I don't know why I like Pacman so much. I mean it's clear that I do like it if I dedicated an entire website to everything that is Pacman. I'm not even that good at it, I don't think I've ever gotten past level 10. I think it just started when I was trying to figure out how to program my own version of Pacman years ago in the C language. Then when I started to learn Python I thought it would be fun to program a version in that language for some reason. I was very proud of my implementation so I decided that I would share it with people who might be interested. I'm hoping that you can find this website useful, and at the very least a little bit entertaining.

I think I like it so much is because it is just fun to program. You learn a lot of great programming concepts programming Pacman from basic collision detection to a very simple AI. It has most game elements that bigger games have, and I think that it provides a great framework for making your own unique games.
